# Insane University Website

## Description
Insane University is a modern, responsive website showcasing an innovative educational institution. The site features a sleek design with black and orange themes, interactive animations, and a comprehensive overview of university programs, campus life, and student testimonials.

## Features
- **Responsive Design**: Optimized for desktop and mobile devices
- **Smooth Scrolling Navigation**: Seamless navigation between sections
- **Interactive Animations**: Fade-in effects, parallax scrolling, and animated counters
- **Gallery Modal**: Clickable gallery images with captions
- **Contact Form**: Functional contact form with submission handling
- **Particle Effects**: Dynamic background particles for visual appeal
- **Typing Effect**: Animated text in the hero section
- **Animated Gradients**: Shifting gradient backgrounds in sections

## Technologies Used
- HTML5
- CSS3 (with animations and responsive design)
- JavaScript (ES6+)
- No external libraries or frameworks required

## Setup
1. Clone or download the project files.
2. Ensure all files are in the same directory:
   - `index.html`
   - `styles.css`
   - `script.js`
   - Image files: `classroom.jpg`, `graduation.jpeg`, `students.webp`, `university place.webp`
3. Open `index.html` in a modern web browser.

## Usage
- Navigate through the site using the menu links.
- Explore different sections: Home, About, Programs, Gallery, Testimonials, Contact.
- Click on gallery images to view them in a modal.
- Fill out the contact form to send a message (currently shows an alert).
